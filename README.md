# 🌍 Overview

EcoSphere is a full-stack ESG Management Platform designed to simplify how organizations manage sustainability initiatives and governance compliance.

The platform enables companies to monitor environmental impact, improve employee participation, maintain governance policies, and visualize ESG performance through an interactive dashboard.

---

# ❗ Problem Statement

Many organizations struggle to:

* Monitor ESG performance efficiently
* Track carbon emissions and sustainability goals
* Manage employee CSR participation
* Maintain governance policies and compliance
* Generate ESG reports for stakeholders

Most existing solutions are expensive and difficult to customize.

---

# 💡 Our Solution

EcoSphere provides one centralized platform where organizations can:

* Monitor ESG Scores
* Track Environmental Metrics
* Manage CSR Activities
* Improve Employee Engagement
* Handle Governance Policies
* Generate Professional ESG Reports
* Visualize ESG Analytics using interactive dashboards

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/Codingwithpiyush/team-collaboration.git

cd EcoSphere
```

---

## Frontend

```bash
cd Frontend

npm install

npm run dev
```

---

## Backend

```bash
cd Backend

pip install -r requirements.txt

python manage.py migrate

python manage.py runserver
```
